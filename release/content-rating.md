# Google Play — Content rating (IARC) & app category

Play Console → **Policy → App content → Content rating**. Mulai kuesioner, pilih kategori
**"Reference, News, or Educational"** (Surau = aplikasi referensi/edukasi, bukan game,
bukan sosial). Jawaban:

| Pertanyaan kuesioner | Jawaban |
|---|---|
| Kekerasan (violence) | **Tidak** |
| Konten menakutkan (fear) | **Tidak** |
| Seksualitas / ketelanjangan | **Tidak** |
| Bahasa kasar / humor vulgar | **Tidak** |
| Narkoba, alkohol, tembakau | **Tidak** |
| Judi / undian (gambling) | **Tidak** |
| Pengguna dapat berinteraksi / bertukar konten / berkomunikasi? | **Tidak** (catatan & tag penanda bersifat pribadi, tidak dibagikan ke pengguna lain; tanpa chat/sosial) |
| App membagikan lokasi pengguna ke pengguna lain? | **Tidak** |
| Pembelian barang digital (digital purchases)? | **Tidak** |
| Akses internet bebas / browser tak terbatas? | **Tidak** (konten Al-Qur'an & terjemahan terkurasi) |

**Hasil yang diharapkan:** rating **Semua umur / Everyone / PEGI 3 / USK 0**.
